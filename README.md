# Customer Churn Prediction using ANN

## Project Overview
This project focuses on predicting customer churn using an Artificial Neural Network (ANN) built with TensorFlow/Keras.
The model predicts whether a customer is likely to leave the bank based on customer-related features.

## Technologies Used
- Python
- Pandas
- NumPy
- TensorFlow/Keras
- Scikit-learn
- Matplotlib

## Workflow
- Data preprocessing
- One-hot encoding
- Feature scaling
- ANN model building
- Dropout regularization
- EarlyStopping implementation
- Model evaluation and visualization

## Model Architecture
- Input Layer
- Hidden Dense Layers with ReLU activation
- Dropout layers for reducing overfitting
- Output layer with Sigmoid activation

## Results
- Achieved approximately 85% accuracy
- Reduced overfitting using Dropout and EarlyStopping

## Deep Learning Concepts Used
- Artificial Neural Networks (ANN)
- Overfitting and Underfitting
- Regularization
- Binary Classification
- Activation Functions

## Dataset
Customer churn dataset from Kaggle.

## Environment
Project developed and trained using Kaggle Notebook environment.
